# FBD-120 HMI Visualization Dashboard

واجهة تحكم وإدارة تفاعلية لآلة **Fluid Bed Dryer-120** (مجفف السرير المميع)

## 🎯 نظرة عامة

هذا المشروع يوفر واجهة رسومية (HMI) كاملة لآلة التجفيف بالسرير المميع FBD-120 من شركة YEDCO. تم تصميم الواجهة بناءً على الكتالوج الأصلي للآلة وتتضمن جميع الشاشات والعناصر التحكمية المطلوبة.

## 📋 المميزات

### الشاشات المتاحة:

1. **Main Menu** - القائمة الرئيسية للوصول إلى جميع الوظائف
2. **Machine Operation 1/2 و 2/2** - التحكم اليدوي في عناصر الآلة
   - Inlet Damper (بوابة الدخول)
   - Shaking (الهز)
   - Bag Lock (إحكام غلق الأكياس)
   - Filter Seal (إحكام غلق الفلتر)
   - Container Seal (إحكام غلق الحاوية)
   - Purging (التنقية)

3. **Material Charging** - إدارة عملية تحميل المواد
4. **Interlocks 1/2 و 2/2** - مراقبة أنظمة الحماية والأقفال الأمنية
5. **Alarm** - عرض سجل الأخطاء والتنبيهات
6. **Auto Cycle** - التحكم في الدورة التلقائية
7. **Recipe Management** - إدارة الوصفات المحفوظة

### المميزات التقنية:

✅ واجهة تفاعلية مع أزرار تتغير ألوانها عند التفعيل (أخضر = مفعل، رمادي = معطل)
✅ شعار شركة YEDCO في الزاوية العلوية
✅ أسماء المهندسين المسؤولين عن البرمجة
✅ تنقل سلس بين جميع الشاشات
✅ تصميم يطابق الكتالوج الأصلي بالضبط

## 🚀 البدء السريع

### المتطلبات:
- Node.js 22+
- pnpm 10+

### التثبيت:

```bash
# استنساخ المستودع
git clone https://github.com/abdulrhmanalrimi/hmi_FBD1.git
cd hmi_FBD1

# تثبيت المكتبات
pnpm install

# تشغيل خادم التطوير
pnpm dev
```

الواجهة ستكون متاحة على: `http://localhost:3000`

### البناء للإنتاج:

```bash
pnpm build
```

### النشر على GitHub Pages:

```bash
./deploy-gh-pages.sh
```

## 🔧 التطوير

### إضافة شاشة جديدة:

1. أضف دالة جديدة في `Home.tsx`
2. أضف حالة جديدة في `currentScreen`
3. أضف زر في القائمة الرئيسية للوصول إلى الشاشة الجديدة

### تخصيص الألوان:

عدّل متغيرات CSS في `client/src/index.css`:

```css
:root {
  --primary: #your-color;
  --background: #your-color;
  /* ... */
}
```
